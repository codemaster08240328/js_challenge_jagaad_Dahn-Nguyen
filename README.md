# jagaad-fe-challenge

## How to run locally

```bash
# install dependencies
$ yarn install

# serve with hot reload at localhost:3000
$ yarn dev

```

## Production build using Docker.

```bash

$ docker build -t jagaad-fe .
$ docker run -it -p 3000:3000 jagaad-fe

```

You can see the project running on localhost:3000
